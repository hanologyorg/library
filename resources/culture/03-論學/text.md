# 論學

《郁文華章─中華文化學與教資源套》：「選篇分析」編號︰3
論學論語
1.子曰：「學而時習之，不亦說乎？有朋自遠方來，不亦樂乎？人
不知而不慍，不亦君子乎？」（《學而》第一）
2.子曰：「學而不思則罔，思而不學則殆。」（《為政》第二）
3.子曰：「學如不及，猶恐失之。」（《泰伯》第八）
4.孔子曰：「生而知之者上也；學而知之者次也；困而學之，又其
次也；困而不學，民斯為下矣。」（《季氏》第十六）
5.子曰：「由也！女聞六言六蔽矣乎？」
對曰：「未也。」
「居！吾語女：好仁不好學，其蔽也愚；好知不好學，其蔽也
蕩；好信不好學，其蔽也賊；好直不好學，其蔽也絞；好勇不
好學，其蔽也亂；好剛不好學，其蔽也狂。」（《陽貨》第十七）
6.子夏曰：「博學而篤志，切問而近思，仁在其中矣。」（《子張》
第十九）
一、簡析
儒家重視人的品德修養，而品德修養源於學。學習、教育是孔子常常談
到的話題，本篇從《論語》中輯錄了孔子對學習的看法。
‧好學成德
孔子認為只要有修德成君子的志向，就能從學習過程中得到喜悅。最1
初，因時時學習，所學者熟，而心中喜悅。至學有所成，招引朋友遠道
而來，與己講習，是快樂的更高層次。最後，學習既有成績，即使別人
不知道我的才具，也不憂愁，也能自得其樂。
‧學思並重
讀書而不思索，昏而無所得；空想而不學習，則危而不安。學習要有所
得，要扎實，必須學思並重。
‧終身不休
孔子認為學習是自發和積極的，而且終身不休。學習之初，積極而恐怕
追趕不上；趕上了，又生怕丟掉，要努力保持。
‧主動求學
對於「生而知之者」，學習自是得心應手，但對一般人來說，他們都需
要靠學習來掌握知識。孔子十分欣賞那些「學而知之」的人，因為他們
主動求學；「困而學之」的人次之，因為他們遇到困難才學，過於被動；
至於「困而不學」的人，無所用心，則屬於最下等了。
‧學以明理
孔子和子路討論「六言六蔽」，旨在解釋學習的作用是明理，不學習就
無以明理。即使你立心為善，做到「仁、知、信、直、勇、剛」，倘不
學習，後果是陷於「愚、蕩、賊、絞、亂、狂」。
‧仁在其中
學習如果要靠近或達到仁的境界，就要先訂立方向，在廣泛學習中堅守
自己的志趣；遇到困難，要向人家發問請益，還要融會貫通，多考慮當
前的問題，把學問應用到現實生活裏。2
